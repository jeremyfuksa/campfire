# Suggested Commands
- `npm create vite@latest` (React + TypeScript) – bootstrap the bundler needed to serve this UI since none exists yet.
- `npm install` inside the generated Vite project – pull React/runtime deps.
- `npm run dev` – start the Vite dev server once configured.
- `npm run build` – produce production assets via Vite when ready to deploy.
- `node scripts/export-tokens.js` – regenerate `figma-tokens.json` from the CSS variables for use in Figma's Tokens Studio plugin.
