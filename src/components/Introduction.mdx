## Getting Started

### Installation

\`\`\`bash
npm install @jeremyfuksa/campfire
\`\`\`

### Usage

\`\`\`tsx
import { Button, DataTable, Autocomplete } from '@jeremyfuksa/campfire';

function App() {
  return (
    <div>
      <Button variant="default">Click me</Button>
    </div>
  );
}
\`\`\`

### CLI Tool

Campfire includes a CLI tool for quick component setup:

\`\`\`bash
# Initialize Campfire in your project
npx campfire init

# List all available components
npx campfire list

# Add specific components
npx campfire add button card input
\`\`\`

## Design Tokens

Campfire uses a comprehensive token system for colors, spacing, typography, and more. All tokens are CSS custom properties that can be customized for your brand.
