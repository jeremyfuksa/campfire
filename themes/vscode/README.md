# Campfire Theme for VS Code

A warm, earthy color theme inspired by gathering around a campfire. Part of the [Campfire Design System](https://jeremyfuksa.github.io/campfire).

Includes both **Light** and **Dark** variants.

## Screenshots

### Campfire Dark

Earthy tones with a deep slate background. Comfortable for long coding sessions with warm syntax highlighting.

### Campfire Light

Clean and warm with a natural palette. High contrast without harshness.

## Features

- Semantic syntax highlighting
- Optimized for React, TypeScript, JavaScript, HTML, CSS
- Warm, earthy color palette derived from the Campfire Design System tokens
- Reduced eye strain with muted tones
- Git decoration colors
- Full terminal ANSI color integration
- Bracket pair colorization support

## Installation

### From VSIX (local)

```bash
code --install-extension campfire-theme-0.1.0.vsix
```

### Manual

1. Copy this folder to your VS Code extensions directory:
   - **macOS/Linux:** `~/.vscode/extensions/campfire-theme`
   - **Windows:** `%USERPROFILE%\.vscode\extensions\campfire-theme`
2. Restart VS Code
3. `Cmd+Shift+P` (or `Ctrl+Shift+P`) → **Preferences: Color Theme**
4. Select **Campfire Dark** or **Campfire Light**
